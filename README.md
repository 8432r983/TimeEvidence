# Time Evidence App

Ovo je aplikacija za internu evidenciju sati zaposlenika.
Napravio sam ju za vrijeme svoje prakse tokom ljeta 2024. godine.

Aplikacija je napravljena u Qt-u, sa programskim jezicima qml and c++.
